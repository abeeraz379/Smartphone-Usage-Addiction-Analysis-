# Smartphone-Usage-Addiction-Analysis-
Smartphone Usage & Addiction Analysis by ABeer Al-Zebda Using DataSet from Kaggle
# Data link
https://www.kaggle.com/datasets/algozee/smartphone-usage-and-addiction-analysis-dataset
# Data Dictionary
Column Name	Description
- User_ID	: Unique identifier assigned to each individual in the dataset.
- Age	: Age of the individual in years.
- Gender :	Gender of the user (e.g., Male, Female).
- Daily_Screen_Time_Hours :	Total number of hours the user spends on their smartphone per day.
- Social_Media_Hours	: Average daily time spent on social media applications.
- Gaming_Hours	: Average daily time spent playing mobile games.
- Work_Study_Hours : 	Time spent using the smartphone for work or study purposes.
- Sleep_Hours : 	Average number of hours the user sleeps per day.
- Notifications_Per_Day	 : Total number of smartphone notifications received per day.
- App_Opens_Per_Day : 	Number of times the user opens apps on their smartphone per day.
- Stress_Level :	Self-reported stress level of the user (e.g., Low, Medium, High).
- Academic_Work_Impact :	Indicates whether smartphone usage negatively affects academic or work performance.
- Addiction_Level : 	A calculated score or level indicating the degree of smartphone addiction.
- Addicted_Label :	Binary classification showing whether the user is considered addicted to smartphone usage (Yes/No or 0/1).
